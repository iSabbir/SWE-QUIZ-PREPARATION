# Topic 01: Lines of Code (LOC) - Online Food Delivery System

Consider an Online Food Delivery System developed for a restaurant chain. The system accepts orders from customers through a web/mobile interface. Before accepting an order, check if the restaurant is open, verify item availability and check customer address validity for delivery. After placing an order, the system will process online payment, update inventory, notify kitchen staff, track delivery status and generate daily and weekly sales reports. The system interacts with customers through a user interface.

## Major Software Functions Identified:

1. User Interface Module  
2. Order & Inventory Management Module  
3. Sales & Analytics Reporting Module
